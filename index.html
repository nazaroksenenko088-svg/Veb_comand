📱 Автоматизация через Shizuku (`rish`) в Termux

Shizuku дает Termux доступ к ADB (UID 2000) без Root-прав. Выполняй системные команды, управляй пакетами и процессами.

🚀 Шаг 1: Авто-установка `rish` в Termux (Выполнить 1 раз)

pkg update && pkg install curl unzip -y && curl -fL "https://github.com/RikkaApps/Shizuku-API/releases/download/v3.5.0/rish-v3.5.0.zip" -o rish.zip && unzip -o rish.zip && chmod +x rish

⚡ Шаг 2: Вход в режим ADB Shell

./rish

*При первом запуске Shizuku на телефоне покажет всплывающее окно — нажми «Разрешить всегда».

🔥 Полезные команды в режиме `rish` (ADB):

# Посмотреть список всех установленных приложений pm list packages # Заморозить / отключить любое приложение без root pm disable-user --user 0 com.example.app # Выдать любое разрешение приложению pm grant com.example.app android.permission.WRITE_SECURE_SETTINGS # Выключить энергосбережение / ограничения фоновой работы для Termux dumpsys deviceidle whitelist +com.termux

⚡ Конфиг клавиш Termux (`termux.properties`)

Вставляет стрелки, ESC, TAB и CTRL прямо над клавиатурой на смартфоне/планшете:

mkdir -p ~/.termux && echo "extra-keys = [['ESC','|','/','HOME','UP','END','PGUP','DEL'],['TAB','CTRL','ALT','LEFT','DOWN','RIGHT','PGDN','ENTER']]" > ~/.termux/termux.properties && termux-reload-settings
