🔥 Полезные Алиасы и Горячие Клавиши (Linux / Termux)

Добавь это в свой ~/.bashrc или ~/.zshrc на планшете/ноутбуке:

# Быстрые алиасы под работу alias cls='clear' alias update='pkg update && pkg upgrade -y || sudo apt update && sudo apt upgrade -y' alias py='python3' alias g='git status' alias push='git add . && git commit -m "update" && git push' # Приветственный кастомный баннер echo -e "\e[1;34m====================================\e[0m" echo -e "\e[1;32m WELCOME TO CYBER OS TERMINAL \e[0m" echo -e "\e[1;34m====================================\e[0m"

⌨️ Горячие клавиши Терминала:

  • Ctrl + C — Прервать выполнение процесса
  • Ctrl + Z — Отправить процесс в фоновый режим
  • Ctrl + R — Поиск по истории команд
  • Ctrl + L — Очистить экран (аналог clear)
  • Tab — Автодополнение команд и путей

💻 Окружения и Эмуляторы

  • Android: Termux + Termux:X11 / AnLinux (для полноценного GUI Linux)
  • Windows (Ноут): WSL2 (Ubuntu / Debian) + Windows Terminal
  • macOS (Макбук): Homebrew + iTerm2 / zsh + OrbStack (легкая альтернатива Docker)
  • Локальный сервер: python3 -m http.server 8080 в папке проекта для тестирования без инета.
